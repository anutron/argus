iOS share sheet

Argus shortcut URL
iOS Safari does not support PWA share targets directly. Build a Shortcut with this URL to put Argus in the share sheet.
  1. Open the Shortcuts app on iPhone, tap + to create a new shortcut.
  2. Tap the (i) at the bottom of the editor → enable Show in Share Sheet and confirm Receive Text, URLs from Share Sheet is on (without it, the magic variable in step 4 won't appear).
  3. Add action URL, paste the URL above into it.
  4. Tap on the [Shortcut Input] text and delete those characters. With the cursor still in that spot, look at the bar above the keyboard — tap Select Variable (or the + icon labeled "Variable") and pick Shortcut Input from the top of the list. The placeholder turns into a blue capsule. That magic variable carries whatever you share — selected text, a Safari URL, a Notes excerpt — into ?text=….
  5. Add action Open URLs (it picks up the URL from step 3 automatically).
  6. Rename the shortcut to Argus — that's the name shown in the share sheet.
  7. Open the Argus PWA at least once and sign in so the auth token is cached; shared content lands on whatever PWA instance the URL routes to.
